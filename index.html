Basics of Scala

Scala is a general-purpose programming language providing support for both object-oriented programming and functional programming.

Scala...

  • Is a high level programming language
  • Is statically typed
  • Syntax is concise but still readable â€” we call it expressive
  • Supports the object-oriented programming (OOP) paradigm
  • Supports the functional programming (FP) paradigm
  • Code results in .class files that run on the Java Virtual Machine (JVM)
  • Is able to use JAVA libraries
Hello World!

"Hello World" is the first program developers use to test systems. So let's write a function that displays "Hello World!" in Scala:

object Hello extends App { println("Hello, world") }

*Object can store variables and methods. The object name should match the filename.

Variables

Scala has two types of variables:

  • val is an immutable variable that should be preferred
  • var creates a mutable variable, and should only be used when there is a specific reason to use it
Declaring a Variable

In Scala, you typically create variables without declaring their type. This is called type inference.

val x = 1 val s = "a string" val p = new Person("Regina")

However, if you wanted to, you can explicitly declare variables with their types. Here is a list of different types in Scala:

  • Int - Integer
  • Double - Decimal
  • Char - Letters
  • String - Sequence of Chars (words)
  • Boolean - True or False
  • Unit - Has no value
val x: Int = 1 val s: String = "a string" val p: Person = new Person("Regina")
Control Structures

Scala has many control stuctures such as if/else, match expressions, try/catch, for loops and expressions, and while and do/ while. We are going to focus on if/else and for loops and expressions.

If/Else control structure is similar to other languages:

if (test1) { doA() } else if (test2) { doB() } else if (test3) { doC() } else { doD() }

The if/else construct returns a value, so, among other things, you can use it as a ternary operator:

val x = if (a < b) a else b

For Loops and Expressions allows you to efficiently write a loop that needs to execute a specific number of times.

for (arg <- args) println(arg) // "x to y" syntax for (i <- 0 to 5) println(i) // "x to y by" syntax for (i <- 0 to 10 by 2) println(i)
